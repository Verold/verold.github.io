New Embed Widget


Default

The new embed widget is reponsive. No matter what the default size for the embed widget is, its maximum width is equal to the width of its container. As the widget resizes, the content automatically aligns with the center of the iframe. The original aspect ratio is not preserved though. You can resize the browser window to see how the embed widget behaves

<iframe frameborder="0" width="600" height="400" src="http://studio.verold.com/projects/542bf9400fa7cb0200000ef1/embed"></iframe>

Fully Reponsive

If you want the embed widget to adapt to the width of its container and always preserve the aspect ratio, you'll need add some custom code to your site. First you need to wrap each iframe like this:

<div class="embed-wrapper">
  <!-- embed code goes here -->
</div>

You also need to add the following CSS code to your stylesheet:

.embed-wrapper{
  position: relative;
  padding-bottom: 66.66%; /* 2/3 aspect ratio */
  height: 0;
  overflow: hidden;
}

.embed-wrapper iframe {
  position: absolute;
  top:0;
  left: 0;
  width: 100%;
  height: 100%;
}
